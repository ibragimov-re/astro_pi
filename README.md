## Тема проекта:</br>Разработка системы управления и наведения оптических приборов в экваториальных координатах

## Описание проекта:
Проект представляет собой программный комплекс на Python под Orange Pi OS на платформе Orange Pi 3 LTS, предназначенный для точного наведения и сопровождения астрономических объектов в экваториальных координатах (RA/Dec). Система обеспечивает автоматическое позиционирование телескопа или фотокамеры на целевой объект с последующей компенсацией суточного вращения Земли, что критически важно для длительной экспозиции в астрофотографии.  

На основе библиотек astropy и skyfield реализованы астрометрические расчёты, включая преобразование координат, учёт рефракции и поправки на местоположение наблюдателя. Модуль трекинга корректирует движение монтировки в реальном времени, минимизируя дрейф объекта в поле зрения.  

Система поддерживает интеграцию с внешними каталогами (Messier, NGC) или прямое управление шаговыми двигателями.

Проект ориентирован на исследователей и астрофотографов, предлагая открытое и гибкое решение для автоматизации наблюдений.
